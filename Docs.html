System Requirements

Windows 7,8,8.1,10.

Google Chrome 59 or later

Installation

1. Download theCybex-Media-Player installer for Windows.

2. Once it is downloaded, run the installer (Cybex Media Player {version} Setup.exe). This will only take a few seconds.

3. By default, Cybex Media Player is installed under C:\Program Files\Cybex Media Player.

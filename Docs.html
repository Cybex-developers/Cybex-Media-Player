System Requirements

Windows XP/Vista/7/8/10

.NET framework 4.5.1 or above if using Windows Vista/XP/7

Installation

1. Download theCybex-Media-Player installer for Windows.

2. Once it is downloaded, run the installer (Cybex Media Player {version} Setup.exe). This will only take a minute.

3. By default, Cybex Media Player is installed under C:\Program Files (x86)\Cybex Media Player.

Download the Cybex Media Player Installer.
Run the installer.
The installer will guide yout through the installtion of the software.

Image Viewer

Supported File types


It currently supports jpeg, gif, png, apng, svg, bmp, bmp ico and png ico.

Editing Filters


You can apply the following filters to your image and you can adjust their values too.

1. Brightness
2. Contrast
3. Blur-point
4. Grayness
5. Hue-rotation
6. Invert
7. Opacity
8. Saturate
9. Sepia

Rotating Image


You can also rotate your image to any degree you want.

Zoom Options:


You can adjust the zoom level of your image
By default it is fixed to normal
You can also fit the image according to the size of viewer window.



Audio Listener

Supported File Types


It currently supports wav, mp3 and ogg.

Time options


You can view the total length and current position in an audio file.
You can also change the current time of the audio.

Speed


You can fast-forward as well as slow-forward the audio.
Fast Forward can be upto 16 times whereas slow-forward can be as less as 0.0625 times.

Volume


You can change the volume accourding to your needs.

Video Player

Supported File Types


It currently supports WebM, mp4 and ogg.

Time options


You can view the total length and current position in a video file.
You can also change the current time of the video.

Speed


You can fast-forward as well as slow-forward the video.
Fast Forward can be upto 16 times whereas slow-forward can be as less as 0.0625 times.

Volume


You can change the volume accourding to your needs.
